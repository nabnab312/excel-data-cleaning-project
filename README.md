# Excel Data Cleaning & Preparation Project

## 📌 Overview

This project demonstrates a complete data cleaning and preparation workflow using Microsoft Excel. The dataset contains raw business transaction records with inconsistent formatting, missing values, and data quality issues.

The objective was to transform the raw dataset into a clean, structured, and analysis-ready dataset that can be used for reporting and business decision-making.

---

## 📊 Dataset

- Records: 40+ business transactions
- Tool: Microsoft Excel
- Data Type:
  - Client Information
  - Contact Person
  - Department
  - Region
  - Payment Method
  - Revenue
  - Profit
  - Profit Margin

## 📈 Data Cleaning Process

### 1. Data Inspection

- Reviewed raw data
- Identified inconsistent formatting
- Checked missing values
- Identified invalid entries

### 2. Data Cleaning

- Removed unnecessary spaces
- Standardized company names
- Corrected text capitalization
- Fixed inconsistent formats
- Replaced invalid values

### 3. Data Validation

- Verified payment methods
- Checked missing revenue values
- Validated profit calculations
- Ensured data consistency

### 4. Data Preparation

- Organized structured tables
- Calculated Profit Margin using Excel formulas
- Prepared clean dataset for further analysis
